# myboilerplate2021

-   Pug、Sass(Dart-Sass)、TypeScript のコンパイル
-   JavaScript のバンドル
-   Babel でトランスパイル

## 準備

```
$ git clone https://github.com/takeshisakuma/myboilerplate2021.git
$ cd myboilerplate2021
$ npm ci
```

## 使用

### Web 制作時

```
$ npm run watch:all
```

### コード整形時

```
$ npm run format:all
```

### 公開前

```
$ npm run prepare:all
```

dist 配下のファイルをそのままアップロード
